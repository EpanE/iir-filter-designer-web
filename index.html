How to Use

  1. Click Start Mic and allow microphone access.
  2. Put on headphones to prevent feedback loops.
  3. Adjust Highpass / Lowpass and Order. Toggle Apply Notch for 50/60 Hz hum.
  4. Use Start Recording → Stop Recording → Save Recordings to download WAV files (raw & filtered).
  5. Pick a WAV save rate: Auto (native) or 16k for speech.

Troubleshooting

  • No audio? Ensure HTTPS, allow mic permissions, and unmute the tab.
  • Cutoff warnings show when values exceed Nyquist (fs/2).
  • On iOS, tap Save again if the download didn’t appear.

Privacy: All processing & recording happen in your browser; nothing is uploaded.
